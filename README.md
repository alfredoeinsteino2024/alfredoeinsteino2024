## 🏆 Latest Project - ChainEngineers

> **Dev3Pack Global Hackathon 2026 - Solana Track**

A Solana-powered embedded payment terminal for everyday merchants in emerging markets like Nigeria. Built as a **Digital Twin simulator** in C with SDL2, communicating with a Node.js backend and confirming real transactions on **Solana Devnet**.

- 🖥️ Terminal UI in C + SDL2 - state machine, 6 screens, real QR code rendering
- 🌐 Raw Winsock2 TCP sockets - no external networking libraries
- ⚡ Real Solana Pay QR code generated and displayed live
- 🔗 Real transactions confirmed on Solana Devnet
- 🧵 Multithreaded with SDL threads + mutexes

🎥 **Demo:** https://www.loom.com/share/e6a757cfbf9748fab841fae9ae3bcf45
📁 **Repo:** https://github.com/alfredoeinsteino2024/chainengineers
