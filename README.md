## 🏆 Latest Project - ChainEngineers Midnight Terminal
> **MLH Midnight Hackathon 2026 - DeFi Track**

A privacy-preserving embedded payment terminal for small merchants in Nigeria.
Built as a Digital Twin simulator in C with SDL2, connected to a Node.js backend
and the Midnight Network for confidential on-chain payments using zero-knowledge proofs.

- 🖥️ Terminal UI in C + SDL2 - state machine, 8 screens, real QR code rendering
- 🌐 Raw Winsock2 TCP sockets - no external networking libraries
- 🔐 ZK commitment scheme - amount and merchant identity never touch the chain
- ⚡ Real midnight: URI QR code generated and displayed live
- 📜 Compact ZK smart contract with three payment circuits
- 🧵 Multithreaded with SDL threads and mutexes
- 🇳🇬 Nigerian Naira to DUST token conversion

🎥 **Demo:** https://www.facebook.com/share/v/1BV5oQZhwY/
📁 **Repo:** https://github.com/alfredoeinsteino2024/chainengineers-midnight
